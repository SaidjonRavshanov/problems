Quyidagi savollarga matn ko'rinishadi javob bering :
1. SSD bn HDD farqini yoritib bering.
2. Dastur deganda nmani tushunasiz ?

javob:
asslomu aleykum vazifalarda o'zimning fikrimni ham qo'shimcha qilib ketyapman.
1 savol javobi__________________________________________________________
SSD(Solid State Drive)
elektron chiplar orqali malumotlar salanadi, harakatlanuvchi qismlari yo'q
hajmi kichik 
tezlik yuqori
narxi qimmatroq
shikastlanishga kamroq moyil HDD ga nisbatan chunki bunda yuqorida aytganimizdek mexanik qismi yo'q
....................................................................................................
HDD (Hard Disk Drive)
o'zining mexanik qismi mavjud, malumotlarni saqlash uchun aylanadigan disk mavjud
ya'ni diskdan 5 mm uzoqlikdagi metal qalamcha undagi malumotlarni qidiradi
hajmi katta ya'ni ko'p malumot sig'dira oladi
tezlik past
narxi azronroq 
shikaslanishga ko'proq moyil chunki qalamcha aylanadigan diskka tegish extimoli yuqori

2.savol javobi_______________________________________________________________________
Dastur bu kompyuter bajarishi uchun dasturchilar tomonidan beriladigan buyruqlar ketma ketligi
ya'ni dastur bir qonuniyat asosida ishlaydi.
kompyuterimiz dastursiz umuman ishlamaydi , chunki kompyuterda operatsion tizimlarda ishlaymiz operatsion tizmlar esa dasturlardir
kompyuterda bizning ko'zimizga ko'ringan narsalar dasturlar sarasiga kiradi, xar xil o'yinla , media player vaho kazo
uni nazorat qiladigan esa dastur kodlari deyiladi.

c,c++ ,c# , java , python.

c#       esa c lar oilasiga kiradi asosida c/c++ yotadi java bilan deyarli bir xil sintaktikaga ega 

java     jvm ga asoslanan dasturlash tili , java kodini jvm ishga tushuradi.
